# 🌍 Travel & Food Insights Platform

This project is a web-based analytics platform that generates city-level travel insights using real-time weather data and a custom scoring model.

## Features
- Real-time weather data integration (OpenWeather API)
- Multi-factor scoring (Travel + Food Index)
- Composite Travel Indicator
- AI-style insight generation

## Tech Stack
- HTML / CSS / JavaScript
- API Integration
- GitHub Pages (deployment)
